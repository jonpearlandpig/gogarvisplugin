# gogarvis-core

This package contains the core logic, models, and utilities for GARVIS governance, authority, routing, verification, intelligence, and sovereignty. It is framework-agnostic and can be used by adapters for different platforms (e.g., OpenClaw, FastAPI, etc).

## Contents
- Core RBAC and governance logic
- Shared models and types
- Utilities for audit logging, authority checks, etc.

## Usage
Import and use in your adapter or backend implementation.
